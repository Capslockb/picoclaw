## 🧠 PicoClaw for Orange Pi

**PicoClaw OrangePi Edition** is an ultra-lightweight AI agent designed specifically for **single-board computers (SBCs)** such as:

- Orange Pi 5  
- Orange Pi RV2 (RISC-V)  
- Orange Pi Zero series  
- Lichee RV Nano  
- other ARM / RISC-V Linux boards

Built in **pure Go**, PicoClaw runs a complete AI agent environment with:

- **<10MB RAM usage**
- **~1 second startup**
- **single static binary**
- **works on RISC-V, ARM64, MIPS and x86**

This makes it ideal for:

- edge AI nodes  
- home servers  
- automation hubs  
- SBC clusters  
- embedded assistants  
- robotics / IoT controllers
